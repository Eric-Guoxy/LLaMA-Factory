The given operation \( z \otimes z = z^2 + 40 \) can be written as follows: \[ z \otimes z = (a + bi) \otimes (a + bi) = a \cdot a + b \cdot b \cdot i = a^2 + b^2 i \] On the other hand, \( z^2 = (a + bi)^2 = a^2 - b^2 + 2abi \). So the equation \( z \otimes z = z^2 + 40 \) becomes: \[ a^2 + b^2 i = (a^2 - b^2 + 2abi) + 40 \] By separating the real and imaginary parts, we get two equations: 1. Real part: \( a^2 = a^2 - b^2 + 40 \) 2. Imaginary part: \( b^2 = 2ab \) From the imaginary part equation \( b^2 = 2ab \), we can solve for \( b \): \[ b^2 = 2ab \implies b(b - 2a) = 0 \] This gives us two cases: 1. \( b = 0 \) 2. \( b = 2a \) Let's consider each case separately. **Case 1: \( b = 0 \)** Substituting \( b = 0 \) into the real part equation: \[ a^2 = a^2 - 0 + 40 \implies a^2 = a^2 + 40 \] This is a contradiction, so \( b = 0 \) is not a valid solution. **Case 2: \( b = 2a \)** Substituting \( b = 2a \) into the real part equation: \[ a^2 = a^2 - (2a)^2 + 40 \implies a^2 = a^2 - 4a^2 + 40 \implies a^2 = -4a^2 + 40 \implies 5a^2 = 40 \implies a^2 = 8 \] So, \( a = \pm \sqrt{8} = \pm 2\sqrt{2} \).
