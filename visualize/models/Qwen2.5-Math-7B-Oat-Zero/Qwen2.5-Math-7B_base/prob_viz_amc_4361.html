We can use dynamic programming to keep track of the probability of each possible running total after each roll. Let's define \(P[i][j]\) as the probability that after \(i\) rolls, the running total is \(j\). We need to calculate this for \(i = 1, 2, 3, 4\) and \(j = 0, 1, 2, \ldots, 24\) (since the maximum possible total after 4 rolls is \(4 \times 6 = 24\)).
