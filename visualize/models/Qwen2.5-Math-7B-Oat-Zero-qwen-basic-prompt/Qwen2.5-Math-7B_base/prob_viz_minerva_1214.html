To solve this problem, we need to use the Doppler effect for light. The observed wavelength \(\lambda_{\text{obs}}\) of a spectral line is related to its rest wavelength \(\lambda_{\text{rest}}\) by the formula: \[ \lambda_{\text{obs}} = \lambda_{\text{rest}} \left(1 + \frac{v}{c}\right) \] where \(v\) is the speed of the galaxy moving away from us, and \(c\) is the speed of light. Given: - \(\lambda_{\text{obs}} = 6784 \, \text{Å}\) - \(\lambda_{\text{rest}} = 6565 \, \text{Å}\) We need to find the ratio \(v/c\). Rearranging the Doppler formula to solve for \(v/c\): \[ \frac{v}{c} = \frac{\lambda_{\text{obs}} - \lambda_{\text{rest}}}{\lambda_{\text{rest}}} \] Substituting the given values: \[ \frac{v}{c} = \frac{6784 \, \text{Å} - 6565 \, \text{Å}}{6565 \, \text{Å}} = \frac{219 \, \text{Å}}{6565 \, \text{Å}} = \frac{219}{6565} \] Now we can calculate the value of \(v/c\): \[ \frac{v}{c} = \frac{219}{6565} \approx 0.0334 \] Let's confirm this calculation using Python. ```python # Given values lambda_obs = 6784 # observed wavelength in Å lambda_rest = 6565 # rest wavelength in Å # Calculate v/c v_over_c = (lambda_obs - lambda_rest) / lambda_rest print(v_over_c) ``` ```output 0.03341734693877551 ``` The calculated value of \(v/c\) is approximately \(0.0334\).
