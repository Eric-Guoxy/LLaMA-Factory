To solve this problem, we need to derive the equation of motion for the rotor. The equation of motion for a rotating system is given by Newton's second law for rotation: \[ I \frac{d\omega}{dt} = T - 2B\omega \] Here, \(I\) is the moment of inertia of the rotor, \(\omega\) is the angular velocity, \(T\) is the applied torque, and \(B\) is the frictional coefficient of each bearing. Since there are two bearings, the total frictional torque is \(2B\omega\). This is a first-order linear differential equation. To solve it, we can use the method of integrating factors. The standard form of the differential equation is: \[ \frac{d\omega}{dt} + \frac{2B}{I}\omega = \frac{T}{I} \] The integrating factor for this differential equation is \(e^{\int \frac{2B}{I} dt} = e^{\frac{2Bt}{I}}\). Multiplying both sides of the differential equation by the integrating factor, we get: \[ e^{\frac{2Bt}{I}} \frac{d\omega}{dt} + \frac{2B}{I} e^{\frac{2Bt}{I}} \omega = \frac{T}{I} e^{\frac{2Bt}{I}} \] The left-hand side is the derivative of the product \(\omega e^{\frac{2Bt}{I}}\): \[ \frac{d}{dt} \left( \omega e^{\frac{2Bt}{I}} \right) = \frac{T}{I} e^{\frac{2Bt}{I}} \] Integrating both sides with respect to \(t\), we get: \[ \omega e^{\frac{2Bt}{I}} = \int \frac{T}{I} e^{\frac{2Bt}{I}} dt + C \] The integral on the right-hand side is: \[ \int \frac{T}{I} e^{\frac{2Bt}{I}} dt = \frac{T}{I} \cdot \frac{I}{2B} e^{\frac{2Bt}{I}} + C_1 = \frac{T}{2B} e^{\frac{2Bt}{I}} + C_1 \] So we have: \[ \omega e^{\frac{2Bt}{I}} = \frac{T}{2B} e^{\frac{2Bt}{I}} + C \] Dividing both sides by \(e^{\frac{2Bt}{I}}\), we get the general solution: \[ \omega(t) = \frac{T}{2B} + C e^{-\frac{2Bt}{I}} \] The constant \(C\) can be determined from the initial conditions. If the initial angular velocity at \(t = 0\) is \(\omega_0\), then: \[ \omega(0) = \frac{T}{2B} + C = \omega_0 \implies C = \omega_0 - \frac{T}{2B} \] So the specific solution is: \[ \omega(t) = \frac{T}{2B} + \left( \omega_0 - \frac{T}{2B} \right) e^{-\frac{2Bt}{I}} \] This is the state-determined equation of motion for the angular velocity \(\omega\) as output and the torque \(T\) as input.
