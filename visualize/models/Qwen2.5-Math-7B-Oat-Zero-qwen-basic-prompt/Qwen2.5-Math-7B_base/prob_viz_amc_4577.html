Using the Pythagorean theorem, we can find the height \(h\): \[ 1^2 = h^2 + \left(\frac{a}{2}\right)^2 \implies 1 = h^2 + \frac{a^2}{4} \implies h^2 = 1 - \frac{a^2}{4} \implies h = \sqrt{1 - \frac{a^2}{4}} = \frac{\sqrt{4 - a^2}}{2} \] The area \(A\) of the trapezoid is given by the formula: \[ A = \frac{1}{2} \times (\text{sum of the bases}) \times \text{height} = \frac{1}{2} \times (a + 2a) \times \frac{\sqrt{4 - a^2}}{2} = \frac{3a \sqrt{4 - a^2}}{4} \] To find the maximum area, we need to find the value of \(a\) that maximizes the area.
