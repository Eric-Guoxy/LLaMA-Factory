To find the original price of the shirt, we need to determine the price before the 60% discount was applied. If the sale price is $14.40 and this is 40% of the original price (since 100% - 60% = 40%), we can set up the following equation: \[ \text{Sale Price} = \text{Original Price} \times 0.40 \] Given that the sale price is $14.40, we can plug this into the equation: \[ 14.40 = \text{Original Price} \times 0.40 \] To find the original price, we can divide both sides of the equation by 0.40: \[ \text{Original Price} = \frac{14.40}{0.40} \] Let's calculate this using Python to ensure accuracy. ```python # Given sale price and discount percentage sale_price = 14.40 discount_percentage = 60 remaining_percentage = 100 - discount_percentage # which is 40% # Calculate the original price original_price = sale_price / (remaining_percentage / 100) print(original_price) ``` ```output 36.0 ``` The original price of the shirt was \(\boxed{36}\) dollars.
